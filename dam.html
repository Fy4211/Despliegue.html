Descripción del Ciclo

El Ciclo Formativo de Grado Superior en Desarrollo de Aplicaciones Multiplataforma (DAM) forma a profesionales especializados en el desarrollo de aplicaciones para diferentes plataformas, desde sistemas móviles hasta aplicaciones de escritorio. Los estudiantes aprenderán a diseñar, implementar, y mantener aplicaciones en entornos tanto locales como distribuidos, asegurando la funcionalidad y la experiencia de usuario en diversos dispositivos y sistemas operativos.

<

Este ciclo es ideal para quienes deseen desarrollar aplicaciones que funcionen en múltiples plataformas, desde dispositivos móviles hasta ordenadores de sobremesa.

Módulos Profesionales

Sistemas Informáticos

Este módulo se centra en los sistemas operativos, hardware y software, permitiendo a los estudiantes comprender cómo funcionan los entornos de desarrollo en distintas plataformas.

<

Programación

Los estudiantes aprenderán a programar aplicaciones en lenguajes orientados a objetos, utilizando tecnologías como Java, C#, o Python para el desarrollo de software multiplataforma.

Desarrollo de Interfaces

<

Aprenderás a diseñar interfaces gráficas de usuario (GUI) para aplicaciones de escritorio, móviles y web, asegurando una experiencia de usuario atractiva y funcional.

<

Acceso a Datos

Se enseñará a gestionar bases de datos y a integrar la interacción con las mismas en aplicaciones, asegurando la eficiencia y seguridad en el tratamiento de datos.

<

Desarrollo en Entornos Multiplataforma

Este módulo se centra en la creación de aplicaciones que funcionan en diferentes sistemas operativos y dispositivos, asegurando la portabilidad del código.

Servicios y Procesos

Aprenderás a diseñar aplicaciones que trabajan en segundo plano, administran procesos y servicios, y que interactúan con otros sistemas distribuidos.

<

Salidas Profesionales

  • Desarrollador de Aplicaciones Multiplataforma: Crearás aplicaciones que funcionen en múltiples sistemas operativos, desde Windows hasta Android o iOS.
  • Desarrollador de Software de Escritorio: Te especializarás en la creación de software específico para sistemas operativos como Windows, macOS o Linux.
  • Desarrollador de Aplicaciones Móviles: Diseñarás y crearás aplicaciones para dispositivos móviles, incluyendo Android y iOS.
  • Administrador de Bases de Datos: Serás el encargado de gestionar, diseñar y optimizar bases de datos que interactúan con aplicaciones multiplataforma.
  • Consultor de Soluciones Multiplataforma: Asesorarás a empresas sobre cómo optimizar sus aplicaciones para que sean compatibles con múltiples entornos tecnológicos.

Requisitos de Acceso

Para acceder al ciclo de DAM es necesario cumplir uno de los siguientes requisitos:

<
  • Poseer el título de Bachillerato.
  • Haber superado un Ciclo Formativo de Grado Medio.
  • Tener un título de Técnico o Técnico Superior de Formación Profesional.
  • Haber superado la prueba de acceso a ciclos formativos de Grado Superior.
<

Duración del Ciclo

El ciclo tiene una duración de 2 años, con un total de 2000 horas lectivas, incluyendo prácticas en empresas (FCT).
