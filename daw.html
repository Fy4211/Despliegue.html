Descripción del Ciclo

El Ciclo Formativo de Grado Superior en Desarrollo de Aplicaciones Web (DAW) está diseñado para formar profesionales en la creación, desarrollo, y mantenimiento de aplicaciones web. Los estudiantes adquirirán conocimientos tanto en el frontend como en el backend, lo que les permitirá gestionar todas las fases de un proyecto de desarrollo web.

Este ciclo está orientado a quienes deseen trabajar en el ámbito de las tecnologías de la información, desarrollando aplicaciones que cumplan con los estándares de usabilidad y accesibilidad.

Módulos Profesionales

Programación

Este módulo introduce a los estudiantes en la programación básica y orientada a objetos, utilizando lenguajes como Java, C#, o Python.

Desarrollo Web en Entorno Cliente

Aprenderás a diseñar interfaces interactivas utilizando tecnologías como HTML5, CSS3, y JavaScript, creando aplicaciones que funcionen en cualquier navegador.

Desarrollo Web en Entorno Servidor

Este módulo te permite crear aplicaciones dinámicas y bases de datos interactivas usando lenguajes como PHP, Python, y bases de datos SQL.

Despliegue de Aplicaciones Web

Se abordan técnicas para el despliegue y mantenimiento de aplicaciones web, incluyendo servidores, dominios y la administración de sistemas.

Diseño de Interfaces Web

Los estudiantes aprenderán a diseñar interfaces atractivas y funcionales, centradas en la experiencia de usuario (UX) y la usabilidad (UI).

Empresa e Iniciativa Emprendedora

Este módulo enseña los principios básicos de la creación de una empresa, desde la idea inicial hasta la gestión y comercialización del producto o servicio.

Salidas Profesionales

  • Desarrollador Web Frontend: Especialista en la creación de interfaces de usuario y experiencia web, utilizando tecnologías como HTML, CSS y JavaScript.
  • Desarrollador Web Backend: Responsable del desarrollo del lado servidor de las aplicaciones web, utilizando lenguajes de programación y bases de datos.
  • Administrador de Bases de Datos: Encargado de diseñar, implementar y gestionar bases de datos seguras y optimizadas para el rendimiento.
  • Consultor Web: Asesor especializado en soluciones web para empresas, incluyendo la mejora de rendimiento, seguridad y accesibilidad de sitios web.
  • Gestor de Proyectos Web: Profesional encargado de la planificación, ejecución y supervisión de proyectos de desarrollo web.

Requisitos de Acceso

Para acceder al ciclo de DAW es necesario cumplir uno de los siguientes requisitos:

  • Poseer el título de Bachillerato.
  • Haber superado un Ciclo Formativo de Grado Medio.
  • Tener un título de Técnico o Técnico Superior de Formación Profesional.
  • Haber superado la prueba de acceso a ciclos formativos de Grado Superior.

Duración del Ciclo

El ciclo tiene una duración de 2 años, con un total de 2000 horas lectivas, incluyendo prácticas en empresas (FCT).
