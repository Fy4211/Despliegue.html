Herramientas Frontend

HTML5

El estándar básico para estructurar el contenido web, permitiendo la creación de páginas dinámicas y accesibles.

CSS3

Lenguaje de diseño utilizado para describir la presentación de un documento escrito en HTML, incluyendo colores, diseños y fuentes.

JavaScript

Lenguaje de programación que permite la creación de contenido dinámico, interactivo y funciones complejas en el navegador.

React.js

Biblioteca JavaScript utilizada para construir interfaces de usuario interactivas y reutilizables, centradas en el desarrollo de SPA (Single Page Applications).

Vue.js

Framework JavaScript progresivo que se utiliza para construir interfaces de usuario interactivas y aplicaciones web dinámicas.

Bootstrap

Un framework CSS para el desarrollo rápido de sitios web responsivos y móviles, con plantillas de diseño y componentes predefinidos.

Tailwind CSS

Un framework de CSS que proporciona clases de utilidad para construir interfaces de usuario personalizadas de forma rápida y eficiente.

Angular

Framework basado en TypeScript para la creación de aplicaciones web robustas y escalables, con un enfoque en la arquitectura modular.

Herramientas Backend

Node.js

Entorno de ejecución para JavaScript en el servidor, que permite crear aplicaciones escalables y de alto rendimiento.

Express.js

Framework minimalista de Node.js para la creación de aplicaciones web y APIs robustas y rápidas.

PHP

Lenguaje de programación ampliamente utilizado para el desarrollo de aplicaciones web dinámicas y gestión de bases de datos.

Python (Django, Flask)

Lenguaje de programación versátil utilizado en el backend con frameworks como Django y Flask para la creación de aplicaciones web rápidas y seguras.

Ruby on Rails

Framework de desarrollo web escrito en Ruby, que permite la creación rápida de aplicaciones mediante la convención sobre la configuración.

Java (Spring)

Lenguaje de programación ampliamente utilizado para el desarrollo de aplicaciones empresariales, con frameworks como Spring para crear sistemas escalables y seguros.

MySQL

Un sistema de gestión de bases de datos relacional, utilizado para almacenar y gestionar datos de aplicaciones web.

MongoDB

Base de datos NoSQL que permite almacenar grandes volúmenes de datos no estructurados o semiestructurados, utilizada en aplicaciones modernas.
